Create navigational links within a web site

There are several technique to navigate within a web site. The semantic tag for navigation is the <, nav$> tag. You can navigate through your web site by using relative addressing. For out example we will use the unorder list tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

Navigational menus are placedin your < header > semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should have a an <hx> tag.
