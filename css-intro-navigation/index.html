Create navigational links within a web site

There are several technique to navigate within a web site. The semantic tag for navigation is the <, nav$> tag. You can navigate through your web site by using relative addressing. For out example we will use the unorder list tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

Navigational menus are placedin your < header > semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should have a an <hx> tag. CSS Coding Styles Summary

Browser defaults (no CSS) You have no control over the styling
Inline this is for a single specific element (tag) instance your CSS will be specified in a parameter within the element. This CSS affects ONLY the element tag that contains the specification. style = "rule, [rule;.....]" style = "color, red; width:50px"
Embedded The specification of CSSS as embedded happens in your < head > area. This CSS affects all similar elements on the web page where the CSS is coded. <style> p {color:red; width:50px;} hi {font-style:italic} <style> p {color:red; width:50px;} hi {font-style:italic}
link (stylesheet) This is the preferred method of doing CSS. You place your CSS in a seperate physical file. This file can then be used on ANY web page in your web site. The link to this file will be done in your < head > area. The CSS in this file is similar to embedded EXCEPT there is no style tag
